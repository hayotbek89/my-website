======================================================
  PROFIL RASMINI QO'YISH YO'RIQNOMASI
======================================================

1. O'zingizning rasmingizni shu "images" papkasiga nusxalang.

2. Rasmni "profile.jpg" deb nomlang.
   (Masalan: hayotbek.jpg  →  profile.jpg  deb o'zgartiring)

3. Saytni brauzerda yangilang (F5) — rasm avtomatik chiqadi!

------------------------------------------------------
Qo'llab-quvvatlanadigan formatlar:
   ✅ profile.jpg   (tavsiya etiladi)
   ✅ profile.jpeg
   ✅ profile.png
   ✅ profile.webp

Agar boshqa format ishlatmoqchi bo'lsangiz,
index.html faylida quyidagi qatorni toping va
kengaytmani o'zgartiring:
   src="images/profile.jpg"  →  src="images/profile.png"

------------------------------------------------------
Rasm o'lchami maslahat:
   📐 500x500 piksel yoki undan katta (kvadrat)
   📦 Hajm: 2 MB dan oshmasin
